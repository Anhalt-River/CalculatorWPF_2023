1. Получение в числе значения бесконечности. Вылет при попытке присвоения в число нового значения. Вывод: введение ограничения на получение бесконечности

2. Попытка получения корня нуля из числа. Вывод: добавление сообщения об ошибке внутрь класса калькулятора.

3. Выпадение из оков блокировки при получении через MOD 'Не числа'. Вывод: для выведения проблемы следует добавить внутреннее сообщение об ошибке в Mod.

4. Получение факториала из -1. Вывод: введение ограничения на обработку факториалом чисел меньше нуля.

5. При попытке ввести дробное число, а впоследствии и суммировать(или произвести прочие операции) с ним другое число, предыдущее только съедает вводимые после мат. знака цифры. Вывод: произвести работы над нынешней системой запятой/произвести более гибкую технологию установки запятой.

6. При попытке удаления последнего знака введенного дробного числа возникает выпадение за границы имеющегося числа. Вывод: введение правки в перебор массива символов имеющегося числа.

7. При проведении реверсирования числа возникла ошибка округления числа ввиду появления за запятой свыше пятнадцати цифр. Вывод: введение ограничения на округление, округляемое число.
